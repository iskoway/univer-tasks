## Практические задания по Excel

В этой папке находятся файлы с решениями практических заданий по курсу "Базы данных" в MS Excel. Каждый файл соответствует отдельной подтеме или набору упражнений и содержит выполненные шаги по шаблону из методички.

### Соглашение по именованию файлов

Имена Excel-файлов имеют вид  
`<порядок-таблиц>_<подтема>_<диапазон-заданий>.xlsx`

- `01_autocomplete_and_links_1_6.xlsx` — файл для подтемы **«Автозаполнение и ссылки»**, в котором выполнены пункты с 1 по 5 из методички.
